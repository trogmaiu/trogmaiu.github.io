Trogdocker

Uma gui para o docker, o intuito desta aplicação é facilitar a gerência, através dos comandos básicos para administração dos containers, através dele é feito procuras, listagem, e execução daqueles que retornem shell, exemplo"OS, UBUNTU".

TrogmaiuVídeos

Um site de conversão de vídeos gratuito. Está ferramenta foi desenvolvida para ajudar os usuários, principalmente aqueles que não tem boa experiência com o computador. Ele converte de formatos variados para o .MP4, podendo converter de 3gp, mov, avi, mpg para o formato mais usado no momento.
